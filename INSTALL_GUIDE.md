# ZytherXOS Installation Guide

## Requirements

* Supported device
* Unlocked bootloader
* Custom recovery installed
* Backup of important data

## Installation Steps

1. Download the latest ZytherXOS build.
2. Transfer the ROM package to your device.
3. Boot into custom recovery mode.
4. Wipe Data, Cache, and System partitions.
5. Flash the Zenith OS package.
6. Reboot the device.
7. Complete the initial setup process.

## Warning

Flashing custom operating systems may cause data loss or device issues if performed incorrectly. Proceed only if you understand the process.
